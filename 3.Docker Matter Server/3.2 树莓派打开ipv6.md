## Verify IPv6 is Enabled on RPI5 System

On Linux, you can check with sysctl net.ipv6.conf.all.disable_ipv6 (should be 0 if enabled)

~~~
# sysctl net.ipv6.conf.all.disable_ipv6
~~~
output
~~~
net.ipv6.conf.all.disable_ipv6 = 0
~~~
